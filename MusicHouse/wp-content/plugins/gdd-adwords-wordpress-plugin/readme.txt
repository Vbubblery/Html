Linkify predefined words in a post.

== Description ==

GDD Adwords add links on predefined words, based on user options, in all WordPress posts.
It does not affect the original post content.


== Installation ==

1. Download the plugin here: http://wordpress.org/extend/plugins/gdd-adwords-wordpress-plugin/ or search in wp-admin/plugins for "gdd-adwords"
2. Install thourgh WordPress admin panel.
3. Activate the plugin through the 'Plugins' menu in WordPress
4. Go to options, select 'GDD Adwords' and submit your keyword list, link address and other URL specific statements.
